Question-1: What are the different ways to select an element in the DOM?
There are many types of selector like getElementsByClassname, getElementsByID, getElementsByTagName. getElementById('id') : Selects a single element with the specified ID; getElementsByClassName('className') : Returns a collection of all elements with the specified class name; getElementsByClassName('className') : Returns a collection of all elements with the specified class name
Question-2: What is the difference between innerHTML, innerText, and textContent?
innerHTML" retrieves or modifies the complete HTML content within an element, including tags, while "innerText" retrieves only the visible text content,textContent" retrieves all text content within an element.
Question-3: What is event delegation in the DOM?
event delegation" is a technique where you attach a single event listener to a parent element instead of adding listeners to multiple child elements
Question-4: What is event bubbling in the DOM?
the behavior where when an event is triggered on an element, it not only affects that element but also propagates upwards through its parent elements in the DOM hierarchy, allowing each parent element along the way to potentially handle the event as well.
Question-5: How do you create, add, and remove elements using JavaScript?
To add, remove, and modify HTML elements using JavaScript, one can use methods like createElement, appendChild, removeChild, and properties like innerHTML and textContent. For example, to add an element, one can create it using document.createElement and then append it to a parent element using appendChild. To remove an element, one can use removeChild on its parent. To modify an element, one can change its innerHTML or textContent.
